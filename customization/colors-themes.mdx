fzf provides extensive customization options for its appearance, including color schemes, UI styles, borders, and more.

## Style Presets

fzf offers three built-in style presets that configure the overall look and feel. Use the `--style` option to quickly apply a preset:

```bash
fzf --style PRESET
```

### Using Style Presets

```bash
# Use the full preset with a preview window
fzf --style full \
    --preview 'fzf-preview.sh {}' \
    --bind 'focus:transform-header:file --brief {}'

# Combine minimal style with custom options
fzf --style minimal --border --preview 'bat --color=always {}'
```

## Color Customization

fzf's color scheme can be fully customized using the `--color` option. Colors can be specified for individual UI elements.

### Color Syntax

```bash
--color=[ELEMENT]:[COLOR]
```

**Color formats:**
- Named colors: `black`, `red`, `green`, `yellow`, `blue`, `magenta`, `cyan`, `white`
- ANSI color codes: `0` through `255`
- Hex colors: `#RRGGBB` (e.g., `#ff0000`)
- Special value: `-1` (use terminal's default color)

### Color Elements

fzf supports extensive color customization for different UI components:

#### Basic Elements

| Element | Description |
| :--- | :--- |
| `fg` | Foreground color for normal items |
| `bg` | Background color for normal items |
| `hl` | Highlighted substrings in normal items |
| `fg+` | Foreground color for current item |
| `bg+` | Background color for current item |
| `hl+` | Highlighted substrings in current item |
| `gutter` | Background color of gutter on the left |

#### UI Components

| Element | Description |
| :--- | :--- |
| `info` | Info area (match count, etc.) |
| `prompt` | Prompt text |
| `pointer` | Pointer to current item |
| `marker` | Multi-select marker |
| `spinner` | Streaming input indicator |
| `header` | Header text |
| `separator` | Separator between main window and preview |
| `scrollbar` | Scrollbar |

#### Border Elements

| Element | Description |
| :--- | :--- |
| `border` | Border color |
| `label` | Border label |
| `preview-border` | Preview window border |
| `preview-label` | Preview window label |
| `preview-scrollbar` | Preview window scrollbar |
| `list-border` | Result list border (with `--style full`) |
| `list-label` | Result list label |
| `input-border` | Input area border (with `--style full`) |
| `input-label` | Input area label |
| `header-border` | Header area border (with `--style full`) |
| `header-label` | Header area label |

#### Preview Elements

| Element | Description |
| :--- | :--- |
| `preview-fg` | Preview window foreground |
| `preview-bg` | Preview window background |

### Advanced Color Configuration

With `--style full`, you can customize colors for different bordered sections:

```bash
git ls-files | fzf --style full \
    --border --padding 1,2 \
    --border-label ' Demo ' \
    --input-label ' Input ' \
    --header-label ' File Type ' \
    --preview 'fzf-preview.sh {}' \
    --bind 'focus:transform-preview-label:[[ -n {} ]] && printf " Previewing [%s] " {}' \
    --color 'border:#aaaaaa,label:#cccccc' \
    --color 'preview-border:#9999cc,preview-label:#ccccff' \
    --color 'list-border:#669966,list-label:#99cc99' \
    --color 'input-border:#996666,input-label:#ffcccc' \
    --color 'header-border:#6699cc,header-label:#99ccff'
```

### Preview Window Colors

Customize the preview window appearance:

```bash
fzf --height 40% --layout reverse --info inline --border \
    --preview 'file {}' \
    --preview-window up,1,border-horizontal \
    --color 'fg:#bbccdd,fg+:#ddeeff,bg:#334455,preview-bg:#223344,border:#778899'
```

## Border Styles

fzf supports various border styles:

```bash
# Full border
fzf --border

# Specific border types
fzf --border=rounded      # Rounded corners
fzf --border=sharp        # Sharp corners
fzf --border=bold         # Bold lines
fzf --border=double       # Double lines
fzf --border=horizontal   # Top and bottom only
fzf --border=vertical     # Left and right only
fzf --border=top          # Top only
fzf --border=bottom       # Bottom only
fzf --border=left         # Left only
fzf --border=right        # Right only
```

### Border with Labels

```bash
# Add labels to borders
fzf --border=rounded \
    --border-label=' Search ' \
    --border-label-pos=3

# Preview window border with label
fzf --preview 'bat --color=always {}' \
    --preview-window border-rounded \
    --bind 'focus:transform-preview-label:echo " Preview: {} "'
```

## Layout Options

```bash
# Layout direction
fzf --layout=default      # Bottom to top
fzf --layout=reverse      # Top to bottom (recommended)
fzf --layout=reverse-list # Reverse list only, keep prompt at bottom

# Info display
fzf --info=default        # Right side
fzf --info=inline         # Same line as input
fzf --info=inline:' > '   # Inline with custom separator
fzf --info=hidden         # Hide info

# Pointer and marker
fzf --pointer='‚Üí'         # Custom pointer character
fzf --marker='‚úì'          # Custom marker for selected items
fzf --prompt='Search: '   # Custom prompt
```

## Padding and Margin

```bash
# Add padding inside the border
fzf --border --padding 1       # 1 row/column on all sides
fzf --border --padding 1,2     # 1 row (top/bottom), 2 columns (left/right)
fzf --border --padding 1,2,3,4 # Top, right, bottom, left

# Add margin outside the border
fzf --margin 1         # 1 row/column on all sides
fzf --margin 1,2       # 1 row, 2 columns
fzf --margin 1,2,3,4   # Top, right, bottom, left
```

## fzf Theme Playground

The [fzf Theme Playground](https://vitormv.github.io/fzf-themes/) is an interactive web tool where you can:

- Visually design your fzf color scheme
- Preview changes in real-time
- Export the `--color` configuration
- Browse community-contributed themes

Created by [Vitor Mello](https://github.com/vitormv), it's the easiest way to create and test custom themes.


## Saving Your Theme

Add your preferred color scheme to your shell configuration:

```bash title="~/.bashrc or ~/.zshrc"
export FZF_DEFAULT_OPTS="
  --layout=reverse
  --border=rounded
  --info=inline
  --prompt='üîç '
  --pointer='‚Üí'
  --marker='‚úì'
  --color=dark
  --color=fg:#d0d0d0,bg:-1,hl:#5f87af
  --color=fg+:#d0d0d0,bg+:-1,hl+:#5fd7ff
  --color=info:#afaf87,prompt:#d7005f,pointer:#af5fff
  --color=marker:#87ff00,spinner:#af5fff,header:#87afaf"
```

Or use a configuration file:

```bash
export FZF_DEFAULT_OPTS_FILE=~/.fzfrc
```

```txt title="~/.fzfrc"
--layout=reverse
--border=rounded

--color=info:#afaf87,prompt:#d7005f,pointer:#af5fff
```
