fzf provides extensive key binding support, allowing you to customize the behavior of nearly every key. You can bind keys to built-in actions or execute external commands.

## Default Key Bindings

### Navigation

| Key | Action | Description |
|-----|--------|-------------|
| `CTRL-K` / `CTRL-P` | Move up | Move cursor up one line |
| `CTRL-J` / `CTRL-N` | Move down | Move cursor down one line |
| `Up` | Move up | Move cursor up one line |
| `Down` | Move down | Move cursor down one line |
| `CTRL-U` | Half page up | Scroll up half a page |
| `CTRL-D` | Half page down | Scroll down half a page |
| `Page-Up` | Page up | Scroll up one page |
| `Page-Down` | Page down | Scroll down one page |
| `Home` / `CTRL-A` | First | Jump to first item |
| `End` / `CTRL-E` | Last | Jump to last item |

### Selection

| Key | Action | Description |
|-----|--------|-------------|
| `Enter` | Accept | Accept current selection and exit |
| `CTRL-C` / `CTRL-G` / `ESC` | Abort | Cancel and exit with no selection |
| `TAB` | Toggle + Down | Toggle selection and move down (multi-select mode) |
| `Shift-TAB` | Toggle + Up | Toggle selection and move up (multi-select mode) |
| `Alt-Enter` | Print query and accept | Print query and current selection |

### Input Editing

| Key | Action | Description |
|-----|--------|-------------|
| `CTRL-H` / `Backspace` | Backward delete char | Delete character before cursor |
| `CTRL-W` | Backward kill word | Delete word before cursor |
| `Alt-Backspace` | Backward kill word | Delete word before cursor |
| `CTRL-U` | Unix line discard | Delete from cursor to beginning of line |
| `CTRL-K` | Kill line | Delete from cursor to end of line |
| `CTRL-A` | Beginning of line | Move cursor to beginning |
| `CTRL-E` | End of line | Move cursor to end |
| `CTRL-B` / `Left` | Backward char | Move cursor left |
| `CTRL-F` / `Right` | Forward char | Move cursor right |
| `Alt-B` | Backward word | Move cursor to previous word |
| `Alt-F` | Forward word | Move cursor to next word |
| `CTRL-Y` | Yank | Paste the killed text |

### Mouse

| Action | Description |
|--------|-------------|
| Click | Select item |
| Double-click | Accept item |
| Scroll | Scroll list |
| Shift-Click | Toggle selection (multi-select mode) |
| Shift-Scroll | Move selection (multi-select mode) |

## Custom Key Bindings

Use the `--bind` option to create custom key bindings.

### Basic Syntax

```bash
fzf --bind 'KEY:ACTION'
fzf --bind 'KEY:ACTION1+ACTION2'  # Multiple actions
fzf --bind 'KEY1:ACTION,KEY2:ACTION'  # Multiple bindings
```

### Key Names

#### Regular Keys
```bash
# Printable characters
fzf --bind 'a:accept'
fzf --bind 'space:toggle'
fzf --bind ',:toggle'  # Special: comma
fzf --bind '+:select'  # Special: plus
```

#### Control Keys
```bash
fzf --bind 'ctrl-a:select-all'
fzf --bind 'ctrl-d:deselect-all'
fzf --bind 'ctrl-t:toggle-all'
fzf --bind 'ctrl-space:toggle'
fzf --bind 'ctrl-/:toggle-preview'
fzf --bind 'ctrl-\:toggle-wrap'
```

#### Alt Keys
```bash
fzf --bind 'alt-a:select-all'
fzf --bind 'alt-enter:print-query'
fzf --bind 'alt-space:toggle'
fzf --bind 'alt-/:toggle-preview'
```

#### Special Keys
```bash
fzf --bind 'enter:accept'
fzf --bind 'esc:abort'
fzf --bind 'tab:toggle+down'
fzf --bind 'btab:toggle+up'  # Shift-Tab
fzf --bind 'up:preview-up'
fzf --bind 'down:preview-down'
fzf --bind 'left:backward-char'
fzf --bind 'right:forward-char'
fzf --bind 'home:first'
fzf --bind 'end:last'
fzf --bind 'insert:toggle'
fzf --bind 'delete:delete-char'
fzf --bind 'backspace:backward-delete-char'
fzf --bind 'page-up:page-up'
fzf --bind 'page-down:page-down'
```

#### Function Keys
```bash
fzf --bind 'f1:toggle-preview'
fzf --bind 'f2:toggle-sort'
fzf --bind 'f12:abort'
```

#### Mouse Events
```bash
fzf --bind 'left-click:accept'
fzf --bind 'right-click:toggle'
fzf --bind 'double-click:accept'
fzf --bind 'scroll-up:preview-up'
fzf --bind 'scroll-down:preview-down'
```

#### Event Triggers
```bash
fzf --bind 'start:reload(find . -type f)'  # On start
fzf --bind 'load:first'  # After loading
fzf --bind 'focus:transform-preview-label:echo {}'  # On focus change
fzf --bind 'result:first'  # After search result update
fzf --bind 'change:reload(rg {q})'  # On query change
fzf --bind 'backward-eof:abort'  # When backspace at empty query
fzf --bind 'one:accept'  # When only one item matches
fzf --bind 'zero:abort'  # When no items match
```

### Execute Actions

<ParamField path="execute(...)" type="action">
  Execute command and return to fzf. Use `{}` for current item, `{+}` for all selected items.
  
  ```bash
  # Open file in less
  fzf --bind 'f1:execute(less -f {})'
  
  # Delete files
  fzf --multi --bind 'ctrl-d:execute(rm {+})'
  
  # Git add files
  fzf --multi --bind 'ctrl-a:execute(git add {+})'
  ```
</ParamField>

<ParamField path="execute-silent(...)" type="action">
  Execute command silently without showing output.
  
  ```bash
  # Copy to clipboard
  fzf --bind 'ctrl-y:execute-silent(echo {} | pbcopy)'
  
  # Open in background
  fzf --bind 'ctrl-o:execute-silent(open {})'
  ```
</ParamField>

<ParamField path="execute-multi(...)" type="action">
  Execute command for each selected item.
  
  ```bash
  # Process each file individually
  fzf --multi --bind 'ctrl-p:execute-multi(echo "Processing: {}")'
  ```
</ParamField>

<ParamField path="become(...)" type="action">
  Replace fzf process with the command (no return to fzf).
  
  ```bash
  # Open in vim
  fzf --bind 'enter:become(vim {})'
  
  # Open multiple files
  fzf --multi --bind 'enter:become(vim {+})'
  
  # Different actions for different keys
  fzf --bind 'enter:become(vim {}),ctrl-e:become(emacs {})'
  ```
</ParamField>

### Dynamic Actions

<ParamField path="reload(...)" type="action">
  Reload the candidate list with new command.
  
  ```bash
  # Reload process list
  ps -ef | fzf --bind 'ctrl-r:reload(ps -ef)'
  
  # Interactive ripgrep
  fzf --bind 'change:reload(rg -i {q})' --disabled
  
  # Switch between file types
  fzf --bind 'ctrl-f:reload(find . -type f),ctrl-d:reload(find . -type d)'
  ```
</ParamField>

<ParamField path="reload-sync(...)" type="action">
  Reload synchronously (wait for command to complete).
  
  ```bash
  fzf --bind 'ctrl-r:reload-sync(expensive-command)'
  ```
</ParamField>

<ParamField path="preview(...)" type="action">
  Change preview command.
  
  ```bash
  # Dynamic preview
  fzf --bind 'ctrl-/:preview(bat --color=always {})'
  ```
</ParamField>

<ParamField path="change-preview(...)" type="action">
  Change preview command dynamically.
  
  ```bash
  fzf --preview 'cat {}' --bind 'ctrl-b:change-preview(bat --color=always {})'
  ```
</ParamField>

<ParamField path="change-preview-window(...)" type="action">
  Change preview window options.
  
  ```bash
  # Toggle preview position
  fzf --preview 'cat {}' --bind 'ctrl-/:change-preview-window(down|right)'
  
  # Cycle through layouts
  fzf --preview 'cat {}' --bind 'ctrl-/:change-preview-window(down|hidden|)'
  ```
</ParamField>

<ParamField path="change-query(...)" type="action">
  Change the query string.
  
  ```bash
  fzf --bind 'ctrl-space:change-query(TODO)'
  ```
</ParamField>

<ParamField path="change-prompt(...)" type="action">
  Change the prompt string.
  
  ```bash
  fzf --bind 'ctrl-p:change-prompt(New prompt> )'
  ```
</ParamField>

<ParamField path="transform(...)" type="action">
  Transform output of command and execute as actions.
  
  ```bash
  # Conditional action
  fzf --bind 'enter:transform:[[ -d {} ]] && echo become(cd {}) || echo become(vim {})'
  ```
</ParamField>

## Advanced Examples

### Multi-Action Bindings

```bash
# Select and move down
fzf --multi --bind 'space:select+down'

# Toggle all and jump to first
fzf --multi --bind 'ctrl-a:select-all+first'

# Clear query and reload
fzf --bind 'ctrl-x:clear-query+reload(find .)'
```

### Conditional Actions with Transform

```bash
# Different actions based on selection
fzf --bind 'enter:transform:
  if [[ -d {} ]]; then
    echo become(cd {})
  elif [[ -f {} ]]; then
    echo become(vim {})
  fi'

# Dynamic preview based on file type
fzf --bind 'focus:transform-preview-label:echo [ {} ]' \
    --bind 'focus:transform:
      if [[ {} == *.jpg ]] || [[ {} == *.png ]]; then
        echo "change-preview(chafa {})"
      else
        echo "change-preview(bat --color=always {})"
      fi'
```

### Interactive Ripgrep

```bash
rg_prefix='rg --column --line-number --no-heading --color=always --smart-case'
fzf --disabled --ansi \
    --bind "start:reload:$rg_prefix ''" \
    --bind "change:reload:$rg_prefix {q} || true" \
    --bind 'enter:become(vim {1} +{2})' \
    --delimiter : \
    --preview 'bat --color=always {1} --highlight-line {2}' \
    --preview-window 'up,60%,border-bottom,+{2}+3/3,~3'
```

### Git Integration

```bash
# Interactive git file selector
git ls-files | fzf \
  --multi \
  --preview 'git diff --color=always {}' \
  --bind 'ctrl-a:select-all' \
  --bind 'ctrl-d:deselect-all' \
  --bind 'ctrl-g:execute(git add {+})' \
  --bind 'ctrl-r:execute(git reset {+})' \
  --bind 'ctrl-c:execute(git commit)'
```

### File Manager

```bash
fzf --preview 'bat --color=always --style=numbers --line-range=:500 {}' \
    --bind 'ctrl-d:execute(rm {})+reload(find . -type f)' \
    --bind 'ctrl-r:execute(mv {} {}.bak)+reload(find . -type f)' \
    --bind 'ctrl-e:execute($EDITOR {})' \
    --bind 'ctrl-o:execute(open {})' \
    --bind 'ctrl-y:execute-silent(echo {} | pbcopy)'
```

## Placeholders

Use these placeholders in execute actions:

| Placeholder | Description |
|-------------|-------------|
| `{}` | Current item |
| `{+}` | All selected items (space-separated) |
| `{n}` | Item at index n (1-based) |
| `{q}` | Current query string |
| `{..}` | All items in range |
| `{1}`, `{2}`, ... | Fields at index (with `--delimiter`) |
| `{-1}`, `{-2}`, ... | Fields from end |

## Environment Variables in Bindings

These variables are available in execute commands:

| Variable | Description |
|----------|-------------|
| `$FZF_QUERY` | Current query string |
| `$FZF_MATCH_COUNT` | Number of matches |
| `$FZF_SELECT_COUNT` | Number of selected items |
| `$FZF_TOTAL_COUNT` | Total number of items |
| `$FZF_PREVIEW_LABEL` | Current preview label |
| `$FZF_BORDER_LABEL` | Current border label |
