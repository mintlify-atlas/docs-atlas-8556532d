fzf provides extensive color customization through the `--color` option. You can choose from preset themes or create completely custom color schemes.

## Base Themes

fzf includes several base themes:

```bash
# Dark theme (default for dark terminals)
fzf --color=dark

# Light theme (for light terminals)
fzf --color=light

# 16-color theme (uses terminal's 16 ANSI colors)
fzf --color=16
fzf --color=base16

# No colors (black and white)
fzf --color=bw
fzf --color=no
```

## Color Elements

You can customize individual color elements using the format `--color=ELEMENT:COLOR[,ELEMENT:COLOR...]`

## Color Formats

### ANSI Color Numbers (0-255)

```bash
# Basic 16 colors (0-15)
fzf --color=fg:7,bg:0

# Extended 256 colors (0-255)
fzf --color=fg:252,bg:233,hl:110
```

### Named Colors

```bash
# Basic colors
fzf --color=fg:black,bg:white
fzf --color=fg:red,bg:green
fzf --color=fg:yellow,bg:blue
fzf --color=fg:magenta,bg:cyan

# Bright colors
fzf --color=fg:bright-white,bg:bright-black
fzf --color=fg:bright-red,bg:bright-green
fzf --color=fg:bright-yellow,bg:bright-blue
fzf --color=fg:bright-magenta,bg:bright-cyan

# Aliases
fzf --color=fg:gray  # Same as bright-black
fzf --color=fg:grey  # Same as bright-black
```

### Hex Colors (RGB)

```bash
# 24-bit true color
fzf --color=fg:#d0d0d0,bg:#1a1a1a
fzf --color=hl:#5fd7ff,hl+:#87d7ff
```

### Special Values

```bash
# Use terminal's default colors
fzf --color=fg:-1,bg:-1

# Inherit from another element
fzf --color=fg+:fg  # fg+ inherits from fg
```

## Text Attributes

Add text attributes after the color:

```bash
# Bold
fzf --color=hl:bright-cyan:bold

# Italic
fzf --color=header:blue:italic

# Underline styles
fzf --color=fg+:white:underline
fzf --color=fg+:white:underline-double
fzf --color=fg+:white:underline-curly
fzf --color=fg+:white:underline-dotted
fzf --color=fg+:white:underline-dashed

# Other attributes
fzf --color=info:yellow:dim
fzf --color=marker:green:blink
fzf --color=pointer:red:reverse
fzf --color=header:cyan:strikethrough

# Multiple attributes
fzf --color=hl:cyan:bold:underline

# Reset to regular (remove bold)
fzf --color=fg:white:regular

# Strip ANSI codes
fzf --color=fg:strip
```

## Creating Custom Themes

### Step 1: Choose Base Theme

```bash
# Start with a base
fzf --color=dark
# or
fzf --color=light
```

### Step 2: Customize Key Elements

```bash
# Customize matches
fzf --color=dark,hl:#5fd7ff,hl+:#87d7ff

# Customize current line
fzf --color=dark,fg+:#d0d0d0,bg+:#262626
```

### Step 3: Add UI Elements

```bash
fzf --color=dark \
    --color=fg:#bbccdd,bg:#1e1e1e \
    --color=hl:#5fd7ff,fg+:#d0d0d0,bg+:#262626,hl+:#5fd7ff \
    --color=info:#afaf87,prompt:#d7005f,pointer:#af5fff \
    --color=marker:#87ff00,spinner:#af5fff,header:#87afaf
```

### Step 4: Test and Refine

```bash
# Test with preview
fzf --preview 'cat {}' \
    --color=your-custom-colors \
    --preview-window=right:50%:border-left

# Test with borders
fzf --border --color=your-custom-colors,border:#aaaaaa
```

## Tips

<Tip>
  **Testing Colors**: Use a color picker or terminal color chart to find ANSI and hex color values that work well together.
</Tip>

<Tip>
  **Theme Generators**: Check out [fzf Theme Playground](https://vitormv.github.io/fzf-themes/) for interactive theme creation.
</Tip>

<Tip>
  **True Color Support**: Make sure your terminal supports 24-bit true color for hex color codes to work properly. Test with:
  ```bash
  echo $COLORTERM  # Should output 'truecolor' or '24bit'
  ```
</Tip>

<Warning>
  **Performance**: Using extensive ANSI colors (via `--ansi` flag) can impact performance on large datasets. Use sparingly for best performance.
</Warning>
