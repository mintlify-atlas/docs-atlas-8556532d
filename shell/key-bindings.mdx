---
title: Key Bindings
description: CTRL-T, CTRL-R, and ALT-C key bindings for command-line productivity
---

fzf provides three powerful key bindings that enhance your command-line workflow. These bindings are available in bash, zsh, and fish after [setting up shell integration](/shell/setup).

## Overview

| Key Binding | Description | Command Variable |
|-------------|-------------|------------------|
| `CTRL-T` | Paste selected files/directories onto command-line | `FZF_CTRL_T_COMMAND` |
| `CTRL-R` | Paste selected command from history | `FZF_CTRL_R_COMMAND` |
| `ALT-C` | Change to selected directory | `FZF_ALT_C_COMMAND` |

## CTRL-T: File and Directory Search

Paste the selected files and directories onto the command-line.

### How It Works

The list is generated using `--walker file,dir,follow,hidden` option, which:
- Searches for files and directories recursively
- Follows symbolic links
- Includes hidden files

### Basic Usage

```bash
# Type vim and press CTRL-T to search for files
vim <CTRL-T>

# Search within a specific directory
vim ~/projects/<CTRL-T>

# Select multiple files with TAB, then ENTER
rm <CTRL-T>
```

### Customization

<Tabs>
  <Tab title="Custom Command">
    Override the default file walker with a custom command:

    ```bash
    # Use fd for faster searching
    export FZF_CTRL_T_COMMAND='fd --type f --hidden --follow --exclude .git'
    ```
  </Tab>
  <Tab title="Walker Options">
    Configure walker behavior using fzf options:

    ```bash
    export FZF_CTRL_T_OPTS="
      --walker-skip .git,node_modules,target
      --preview 'bat -n --color=always {}'
      --bind 'ctrl-/:change-preview-window(down|hidden|)'"
    ```
  </Tab>
</Tabs>

### Configuration Variables

- `FZF_CTRL_T_COMMAND` - Custom command to generate the file list
- `FZF_CTRL_T_OPTS` - Additional options to pass to fzf

## CTRL-R: Command History Search

Paste the selected command from history onto the command-line.

### Features

- **Toggle Sorting**: Press `CTRL-R` again to switch between relevance and chronological order
- **Raw Mode**: Press `ALT-R` to see surrounding items of a match
  - Use `CTRL-N` and `CTRL-P` to move between matching items only
- **Line Wrapping**: Press `CTRL-/` or `ALT-/` to toggle line wrapping

### Basic Usage

```bash
# Press CTRL-R and start typing to search history
<CTRL-R> git commit

# Navigate with arrow keys or CTRL-N/CTRL-P
# Press ENTER to select and execute
```

### Customization

```bash
# Copy command to clipboard without executing
export FZF_CTRL_R_OPTS="
  --bind 'ctrl-y:execute-silent(echo -n {2..} | pbcopy)+abort'
  --color header:italic
  --header 'Press CTRL-Y to copy command into clipboard'"
```

<Note>
Custom override via `FZF_CTRL_R_COMMAND` is not yet supported and will emit a warning.
</Note>

### Configuration Variables

- `FZF_CTRL_R_OPTS` - Additional options to pass to fzf

## ALT-C: Directory Navigation

Quickly change to a selected directory.

### How It Works

The list is generated using `--walker dir,follow,hidden` option, which:
- Searches for directories recursively
- Follows symbolic links
- Includes hidden directories

### Basic Usage

```bash
# Press ALT-C to search and cd into a directory
<ALT-C>

# Search from a specific starting point
cd /var && <ALT-C>
```

### Customization

<Tabs>
  <Tab title="Custom Command">
    ```bash
    # Use fd to list directories
    export FZF_ALT_C_COMMAND='fd --type d --hidden --follow --exclude .git'
    ```
  </Tab>
  <Tab title="Preview">
    ```bash
    # Show directory tree in preview window
    export FZF_ALT_C_OPTS="
      --walker-skip .git,node_modules,target
      --preview 'tree -C {}'"
    ```
  </Tab>
</Tabs>

### Fish Shell Behavior

<Note>
In fish shell, `CTRL-T` uses the last token on the command-line as the root directory:

```fish
ls /var/<CTRL-T>  # Searches within /var/
```

Use the `$dir` variable in custom commands:

```fish
set -g FZF_CTRL_T_COMMAND "command find -L \$dir -type f 2> /dev/null | sed '1d; s#^\./##'"
```
</Note>

## Display Modes

Configure how fzf appears for each key binding:

```bash
# Individual display modes
export FZF_CTRL_T_OPTS='--tmux bottom,60% --height 60% --border top'
export FZF_CTRL_R_OPTS='--tmux bottom,60% --height 60% --border top'
export FZF_ALT_C_OPTS='--tmux bottom,60% --height 60% --border top'

# Or set globally
export FZF_DEFAULT_OPTS='--height 40% --tmux bottom,40% --layout reverse --border top'
```

## Implementation Details

### Bash

<Tabs>
  <Tab title="Bash 4+">
    ```bash
    # Uses -x flag for execute-on-enter bindings
    bind -m emacs-standard -x '"\C-t": fzf-file-widget'
    bind -m vi-command -x '"\C-t": fzf-file-widget'
    bind -m vi-insert -x '"\C-t": fzf-file-widget'
    ```
  </Tab>
  <Tab title="Bash < 4">
    ```bash
    # Uses complex key sequence for compatibility
    bind -m emacs-standard '"\C-t": " \C-b\C-k \C-u`__fzf_select__`\e\C-e\C-\e(\C-a\C-y\C-h\C-e\e \C-y\ey\C-x\C-x\C-f\C-y\ey\C-_"'
    ```
  </Tab>
</Tabs>

### Zsh

```zsh
# Create custom widgets
zle -N fzf-file-widget
zle -N fzf-history-widget
zle -N fzf-cd-widget

# Bind to keys
bindkey '^T' fzf-file-widget
bindkey '^R' fzf-history-widget
bindkey '\ec' fzf-cd-widget
```

### Fish

```fish
# Bind functions to keys
bind \ct fzf-file-widget
bind \cr fzf-history-widget
bind \ec fzf-cd-widget

# Also bind in insert mode
bind -M insert \ct fzf-file-widget
bind -M insert \cr fzf-history-widget
bind -M insert \ec fzf-cd-widget
```
